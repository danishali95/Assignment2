Ans4:

  1. HTML: Defines the root element of an HTML document.
  2. head: Contains metadata about the HTML document.
  3. title: Sets the title of the HTML document, displayed in the browser's title bar ortab.
  4. body: Contains the visible content of the HTML document.
  5. p: Defines a paragraph of text.
  6. a: Creates hyperlinks to other web pages or resources.
  7. img: Embeds images in the document.
  8. ul: Defines an unordered (bulleted) list.
  9. ol: Defines an ordered (numbered) list.
  10. li: Represents a list item within an ordered or unordered list.
  11. table: Defines a table for organizing data into rows and columns.
  12. tr: Represents a table row.
  13. th: Defines a table header cell.
  14. td: Represents a table data cell.
  15. form: Contains form elements used for user input.
  16. input: Creates input fields for forms, like text boxes, checkboxes, and radio buttons.
  17. button: Defines a clickable button within a form.
  18. textarea: Provides a multiline text input area within a form.
